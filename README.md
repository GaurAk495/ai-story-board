# 🎬 Storyboard AI - Powered by Pollinations.ai

Storyboard AI is an open-source, next-generation web application that transforms a simple text idea into a fully visualized, shot-by-shot cinematic storyboard in seconds. It leverages the raw speed and zero-friction ecosystem of **[Pollinations.ai](https://pollinations.ai)** to instantly synthesize massive 16:9 HD cinematic storyboard panels.

> ✨ Submitted as an official app showcase to the Pollinations AI ecosystem.

## 🌸 Features
- **Instant Image Pipelines:** Every image is generated seamlessly and completely frictionlessly via the **[Pollinations text-to-image API](https://pollinations.ai)**.
- **Visual Aesthetic Selector:** Instantly adapt your storyboard into styles like Anime, Minimalist Stickman, Cinematic Movie, or Franco-Belgian Comic.
- **Micro-Scene Generation:** Breaks your story down into professional screenplay beats, ensuring consistency and strong narrative flow.
- **Director's Editor:** Tweak prompts, override styles, and seamlessly "Reshoot" any panel you don't like via regenerating a completely fresh Pollinations seed.
- **Instant ZIP Export:** Downloads your fully formatted script and all generated HD images directly native in your browser.

## 🚀 Built With
- **[Pollinations.ai](https://pollinations.ai)**: Lightning-fast, free, and open text-to-image API.
- **Next.js 16 (App Router)**: For serverless API execution and seamless React hydration.
- **Tailwind CSS & Framer Motion**: For the buttery-smooth, cinematic dark UI.
- **Zustand**: For global, ultra-fast state management.

## 🛠️ Getting Started
1. Clone the repository.
2. Install dependencies: \`bun install\`
3. Set your \`GEMINI_API_KEY\` in \`.env.local\`.
4. (*Optional*) Set \`POLLINATION_API\` if using a tiered secret auth token.
5. Run the development server: \`bun run dev\`
